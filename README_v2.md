## 🚀 БЫСТРЫЙ СТАРТ

### Вариант 1: Просто приложение (без голоса)

1. Открой `index.html` в браузере
2. Пользуйся! Всё работает локально

### Вариант 2: С ChatGPT Voice (полная мощь!)

1. **Запусти файл `START_ALL.bat`** - он запустит всё автоматически!

Или вручную:

```bash
# Терминал 1: API сервер
cd mcp-server
npm run api

# Терминал 2: MCP сервер для ChatGPT
cd mcp-server
npm start

# Открой index.html в браузере
```

2. **Настрой ChatGPT Desktop** (см. `VOICE_SETUP_FINAL.md`)

3. **Говори с ChatGPT** - задачи появятся в приложении!
